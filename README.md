## Installation

Clone the repo

Install the dependencies:

```bash
npm install
```


Set the environment variables(see src/config/index to know what environment variables this app need):

```bash
cp .env.example .env

# open .env and modify the environment variables (if needed)
```

Run application:
```bash
npm run dev
```

Run tests:
```bash
npm run test
```
